# AGENT_CONTEXT.md

## Project

Mall of America Interactive Sales Deck

---

## Objective

Build a cinematic browser-based sales deck for Mall of America.

This is NOT a traditional website.

This is NOT an admin dashboard.

This is NOT an ecommerce application.

The experience should feel like a premium Digideck presentation combining:

* Apple
* Tesla
* Disney
* Dubai Mall

The deck must drive:

1. Retail Leasing
2. Sponsorship Opportunities
3. Event Bookings

---

## Audience

* Retail Brands
* Luxury Brands
* Sponsors
* Event Organizers
* Corporate Partners

---

## Tech Stack

* Next.js 16
* TypeScript
* Tailwind CSS
* Framer Motion
* GSAP
* Lenis

No backend required.

No authentication required.

No database required.

---

## Design Principles

* Video First
* Luxury UI
* Cinematic Experience
* Minimal Navigation
* High Performance
* Responsive Design

Avoid:

* Dashboard layouts
* Admin style UI
* Generic SaaS designs
* Excessive forms
